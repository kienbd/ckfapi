# CKFAPI

Ruby connector to access the CKF API

## Installation
    gem install 'ckfapi',:git => "https://github.com/kienbd/ckfapi.git"

## Usage

    require 'ckfapi'

    Ckfapi.api_key = 'api-key'
    Ckfapi.api_uri = 'http://your-api-uri'

### Using Ruby on Rails
Put this config in ```config/initializers/ckfapi.rb. ```
